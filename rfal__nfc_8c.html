RFAL NFC device

This module provides the required features to behave as an NFC Poller or Listener device. It grants an easy to use interface for the following activities: Technology Detection, Collision Resollution, Activation, Data Exchange, and Deactivation

This layer is influenced by (but not fully aligned with) the NFC Forum specifications, in particular: Activity 2.0 and NCI 2.0
