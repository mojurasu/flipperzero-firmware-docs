NFC-A T2T Poller Read.

This method sends a Read command to a NFC-A T2T Listener device

Parameters
[in]blockNum: Number of the block to read
[out]rxBuf: pointer to place the read data
[in]rxBufLen: size of rxBuf (RFAL_T2T_READ_DATA_LEN)
[out]rcvLen: actual received data
Returns
ERR_WRONG_STATE : RFAL not initialized or mode not set
ERR_PARAM : Invalid parameter
ERR_PROTO : Protocol error
ERR_NONE : No error
