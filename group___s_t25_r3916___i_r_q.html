◆ st25r3916WaitForInterruptsTimed()

uint32_t st25r3916WaitForInterruptsTimed ( uint32_t  mask,
uint16_t  tmo 
)

Wait until an ST25R3916 interrupt occurs.

This function is used to access the ST25R3916 interrupt flags. Use this to wait for max. tmo milliseconds for the first interrupt indicated with mask mask to occur.

Parameters
[in]mask: mask indicating the interrupts to wait for.
[in]tmo: time in milliseconds until timeout occurs. If set to 0 the functions waits forever.
Returns
: 0 if timeout occured otherwise a mask indicating the cleared interrupts.
