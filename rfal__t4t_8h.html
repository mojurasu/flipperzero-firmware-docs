Detailed Description

Provides convenience methods and definitions for T4T (ISO7816-4)

Author
Gustavo Patricio

This module provides an interface to exchange T4T APDUs according to NFC Forum T4T and ISO7816-4

This implementation was based on the following specs:

  • ISO/IEC 7816-4 3rd Edition 2013-04-15
  • NFC Forum T4T Technical Specification 1.0 2017-08-28
